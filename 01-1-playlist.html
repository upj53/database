플레이리스트 데이터베이스를 구성하는 SQL문 작성하기

테이블 생성하기

PLAYLIST 테이블의 칼럼 정보
순번 칼럼명 자료형 칼럼 상세설정 설명
1 IDX NUMBER 기본키 순서값
2 PLAY_TITLE VARCHAR2(40) 빈값 허용 안됨 곡 제목
3 PLAY_ARTIST VARCHAR2(30) 빈값 허용 안됨 아티스트
4 PLAY_GENRE VARCHAR2(10) 빈값 허용 장르
5 PLAY_LIKE NUMBER 빈값 허용 곡 좋아요 횟수
PLAYLIST 테이블을 칼럼 정보대로 생성하시오.
CREATE TABLE PLAYLIST ( IDX NUMBER PRIMARY KEY, PLAY_TITLE VARCHAR2(40) NOT NULL, PLAY_ARTIST VARCHAR2(30) NOT NULL, PLAY_GENRE VARCHAR2(10) NULL, PLAY_LIKE NUMBER NULL );

자료 입력하기

첫번째 자료를 입력하시오.
INSERT INTO PLAYLIST (IDX, PLAY_TITLE, PLAY_ARTIST, PLAY_GENRE, PLAY_LIKE) VALUES (1, '모르시나요', '조째즈', '발라드', 76260);
두번째 자료를 입력하시오.
INSERT INTO PLAYLIST (IDX, PLAY_TITLE, PLAY_ARTIST, PLAY_GENRE, PLAY_LIKE) VALUES (2, '나는 반딧불', '황가람', '발라드', 144372);
세번째 자료를 입력하시오.
세번째 자료의 좋아요 횟수(PLAY_LIKE)는 빈값 입니다.
INSERT INTO PLAYLIST (IDX, PLAY_TITLE, PLAY_ARTIST, PLAY_GENRE, PLAY_LIKE) VALUES (3, 'like JENNIE', '제니', '댄스', NULL);

자료 수정하기

첫번째 자료를 수정하시오.
IDX가 1인 자료의 좋아요 횟수(PLAY_LIKE)를 77777로 수정하시오.
UPDATE PLAYLIST SET PLAY_LIKE=77777 WHERE IDX=1;
두번째 자료를 수정하시오.
IDX가 2인 자료의 좋아요 횟수(PLAY_LIKE)를 144444로 수정하시오.
UPDATE PLAYLIST SET PLAY_LIKE=144444 WHERE IDX=2;

자료 삭제하기

자료를 삭제하시오.
IDX가 2인 자료를 삭제하시오.
DELETE FROM PLAYLIST WHERE IDX=2;
