자동차 판매실적 데이터베이스를 구성하는 SQL문 작성하기

테이블 생성하기

CAR 테이블의 칼럼 정보
순번 칼럼명 자료형 칼럼 상세설정 설명
1 IDX NUMBER 기본키 순서값
2 CAR_NAME VARCHAR2(40) 빈값 허용 안됨 자동차 이름
3 CAR_BRAND VARCHAR2(20) 빈값 허용 안됨 자동차 브랜드
4 CAR_TYPE VARCHAR2(20) 빈값 허용 자동차 유형
5 CAR_SELL NUMBER 빈값 허용 자동차 판매량
PLAYLIST 테이블을 칼럼 정보대로 생성하시오.
CREATE TABLE CAR ( IDX NUMBER PRIMARY KEY, CAR_NAME VARCHAR2(40) NOT NULL, CAR_BRAND VARCHAR2(20) NOT NULL, CAR_TYPE VARCHAR2(20) NULL, CAR_SELL NUMBER NULL );

자료 입력하기

첫번째 자료를 입력하시오.
INSERT INTO CAR (IDX, CAR_NAME, CAR_BRAND, CAR_TYPE, CAR_SELL) VALUES (1, '쏘렌토', '기아', 'SUV', 8796);
두번째 자료를 입력하시오.
INSERT INTO CAR (IDX, CAR_NAME, CAR_BRAND, CAR_TYPE, CAR_SELL) VALUES (2, '카니발', '기아', 'SUV', 7592);
세번째 자료를 입력하시오.
세번째 자료의 자동차 판매량(PRODUCT_COUNT)은 빈값 입니다.
INSERT INTO CAR (IDX, CAR_NAME, CAR_BRAND, CAR_TYPE, CAR_SELL) VALUES (3, '아반떼', '현대', '준중형 세단', NULL);

자료 수정하기

첫번째 자료를 수정하시오.
IDX가 1인 자동차 판매량(CAR_SELL)을 8900으로 수정하시오.
UPDATE CAR SET CAR_SELL=8900 WHERE IDX=1;
두번째 자료를 수정하시오. IDX가 2인 자동차 판매량(CAR_SELL)을 7600으로 수정하시오.
UPDATE CAR SET CAR_SELL=7600 WHERE IDX=2;

자료 삭제하기

자료를 삭제하시오.
IDX가 2인 자료를 삭제하시오.
DELETE FROM PRODUCT WHERE IDX=2;
