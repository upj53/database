쇼핑몰 데이터베이스를 구성하는 SQL문 작성하기

테이블 생성하기

PRODUCT 테이블의 칼럼 정보
순번 칼럼명 자료형 칼럼 상세설정 설명
1 IDX NUMBER 기본키 순서값
2 PRODUCT_NAME VARCHAR2(40) 빈값 허용 안됨 상품명
3 PRODUCT_PRICE NUMBER 빈값 허용 안됨 판매가격
4 PRODUCT_TYPE VARCHAR2(10) 빈값 허용 상품 분류
5 PRODUCT_COUNT NUMBER 빈값 허용 재고수량
PRODUCT 테이블을 칼럼 정보대로 생성하시오.
CREATE TABLE PRODUCT ( IDX NUMBER PRIMARY KEY, PRODUCT_NAME VARCHAR2(40) NOT NULL, PRODUCT_PRICE NUMBER NOT NULL, PRODUCT_TYPE VARCHAR2(10) NULL, PRODUCT_COUNT NUMBER NULL );

자료 입력하기

첫번째 자료를 입력하시오.
INSERT INTO PRODUCT (IDX, PRODUCT_NAME, PRODUCT_PRICE, PRODUCT_TYPE, PRODUCT_COUNT) VALUES (1, '슬림셔츠 블랙', 29000, '셔츠', 300);
두번째 자료를 입력하시오.
INSERT INTO PRODUCT (IDX, PRODUCT_NAME, PRODUCT_PRICE, PRODUCT_TYPE, PRODUCT_COUNT) VALUES (2, '슬림셔츠 베이지', 29000, '셔츠', 200);
세번째 자료를 입력하시오.
세번째 자료의 좋아요 재고수량(PRODUCT_COUNT)은 빈값 입니다.
INSERT INTO PRODUCT (IDX, PRODUCT_NAME, PRODUCT_PRICE, PRODUCT_TYPE, PRODUCT_COUNT) VALUES (3, '슬림셔츠 블루', 29000, '셔츠', NULL);

자료 수정하기

첫번째 자료를 수정하시오.
IDX가 1인 재고수량(PRODUCT_COUNT)을 250으로 수정하시오.
UPDATE PRODUCT SET PRODUCT_COUNT=250 WHERE IDX=1;
두번째 자료를 수정하시오.
IDX가 2인 재고수량(PRODUCT_COUNT)을 190으로 수정하시오.
UPDATE PRODUCT SET PRODUCT_COUNT=190 WHERE IDX=2;

자료 삭제하기

자료를 삭제하시오.
IDX가 2인 자료를 삭제하시오.
DELETE FROM PRODUCT WHERE IDX=2;
